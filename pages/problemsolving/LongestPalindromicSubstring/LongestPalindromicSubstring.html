Longest Palindromic Substring

This is a classic problem that can be solved using dynamic programming. A palindrome is a string that is spelled the same from both ends. For example, the word "RACECAR" or "MADAM" are palindromes - they are spelled the same regardless of starting from the beginning or starting from the end. Next up, a substring of a given string is a contiguous chunck of the string. For example, "CE", "RACE", "CAR", etc. are all substrings of the string "RACECAR". Notice the distinction from subsequence - a subsequence does not have to be contiguous. For example, "RCR" is a subsequence of the string "RACECAR", but it is not a substring.

With the basic definitions out of the way, we can focus on the given problem, which is to find the longest palindromic substring of a given string. For example, for the input "BANANA", it should output "ANANA", which is the longest substring of this string that is also a palindrome.

Brute Force - Check All Possible Substrings
bool isPalindrome(std::string s, int start, int end) { if (start == end) { return true; } return ((s[start] == s[end]) && isPalindrome(s, start+1, end-1)); } std::string get_lps_using_bruteForce(std::string s) { int len = s.length(); int startAt = 0; int lpf_len = 1; for (int i = 0; i < len-1; i++) { for (int j = i+1; j < len; j++) { if (isPalindrome(s, i, j) && (lpf_len <= j-i+1)) { startAt = i; lpf_len = j-i+1; } } } return s.substr(startAt, lpf_len); }
Recursive Approach
bool isPalindrome(std::string s, int start, int end) { if (start == end) { return true; } return ((s[start] == s[end]) && isPalindrome(s, start+1, end-1)); } std::pair get_lps_using_recursion(std::string s, int startAt, int endAt) { if (startAt == endAt) { return std::make_pair(startAt, 1); } if (isPalindrome(s, startAt, endAt)) { return std::make_pair(startAt, (endAt-startAt+1)); } else { std::pair result1 = get_lps_using_recursion(s, startAt, endAt-1); std::pair result2 = get_lps_using_recursion(s, startAt+1, endAt); if (result2.second > result1.second) { return result2; } else { return result1; } } }
Dynamic Programming
std::string lps(std::string s) { int len = s.length(); int startAt = 0; int paliLen = 1; bool** table = new bool*[len]; for (int i = 0; i < len; i++) { table[i] = new bool[len]; for (int j = 0; j < len; j++) { table[i][j] = false; } } // Palindrome of 1 char for (int i = 0; i < len; i++) { table[i][i] = true; } // Palindrome of 2 chars for (int i = 0; i < len-1; i++) { if (s[i] == s[i+1]) { table[i][i+1] = true; startAt = i; paliLen = 2; } } // Palindrome of 3 chars and up for (int currentLen = 3; currentLen <= len; currentLen++) { for (int i = 0; i <= len-currentLen; i++) { int j = i+currentLen-1; if ((s[i] == s[j]) && (table[i+1][j-1] == true)) { table[i][j] = true; startAt = i; paliLen = currentLen; } } } for (int i = 0; i < len; i++) { delete[] table[i]; } delete[] table; return s.substr(startAt, paliLen); }
